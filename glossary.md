# Technical Glossary

1. **DOM (Document Object Model)**
   - **EN:** A programming interface for web documents that represents the page so that programs can change the document structure, style, and content.
   - **KA:** ვებ-დოკუმენტების პროგრამირების ინტერფეისი, რომელიც წარმოადგენს გვერდის სტრუქტურას, რათა პროგრამებმა შეძლონ მისი სტილის, შინაარსისა და სტრუქტურის შეცვლა.

2. **LocalStorage**
   - **EN:** A web storage web API that allows JavaScript websites to store key-value data in a web browser with no expiration date.
   - **KA:** ვებ-შენახვის API, რომელიც საშუალებას აძლევს JavaScript-ს შეინახოს "key-value" მონაცემები ბრაუზერში დროის ლიმიტის გარეშე.

3. **Event Listener**
   - **EN:** A procedure in JavaScript that waits for an event to occur (like a click or submit) and executes a specific function.
   - **KA:** პროცედურა JavaScript-ში, რომელიც ელოდება კონკრეტული მოვლენის მოხდენას (მაგალითად, კლიკს ან ფორმის გაგზავნას) და ასრულებს შესაბამის ფუნქციას.

4. **Array.prototype.reduce()**
   - **EN:** A higher-order array method that executes a user-supplied callback function on each element of the array, resulting in a single output value.
   - **KA:** მასივის უმაღლესი რიგის მეთოდი, რომელიც ასრულებს ფუნქციას მასივის თითოეულ ელემენტზე და აბრუნებს ერთ საბოლოო მნიშვნელობას (გამოყენებულია სტატისტიკაში).

5. **Asynchronous JavaScript (async/await)**
   - **EN:** Syntactic sugar built on top of Promises, allowing developers to write asynchronous code that looks and behaves like synchronous code.
   - **KA:** სინტაქსური კონსტრუქცია, რომელიც აგებულია Promise-ებზე და საშუალებას იძლევა ასინქრონული კოდი დაიწეროს სინქრონული კოდის მსგავსად (გამოყენებულია API Fetch-ისას).

6. **JSON (JavaScript Object Notation)**
   - **EN:** A lightweight format for storing and transporting data, commonly used when data is sent from a server to a web page.
   - **KA:** მონაცემთა შენახვისა და გადაცემის მსუბუქი ტექსტური ფორმატი, რომელიც ხშირად გამოიყენება სერვერიდან ვებ-გვერდზე მონაცემების გასაგზავნად.

7. **Media Query**
   - **EN:** A CSS technique used to apply styles based on device characteristics, such as screen width or resolution, for responsive design.
   - **KA:** CSS ტექნიკა, რომელიც გამოიყენება სტილების მისანიჭებლად მოწყობილობის მახასიათებლების (მაგ. ეკრანის სიგანის) მიხედვით რესპონსივი დიზაინისთვის.

8. **API (Application Programming Interface)**
   - **EN:** A set of protocols and tools that allows different software applications to communicate with each other.
   - **KA:** პროტოკოლებისა და ხელსაწყოების ნაკრები, რომელიც საშუალებას აძლევს სხვადასხვა პროგრამულ აპლიკაციას დაუკავშირდნენ ერთმანეთს (მაგ. DummyJSON).

9. **Router Guard (Auth Guard)**
   - **EN:** A security mechanism used in web applications to restrict unauthorized users from accessing private routes before login.
   - **KA:** უსაფრთხოების მექანიზმი ვებ-აპლიკაციებში, რომელიც უზღუდავს არავტორიზებულ მომხმარებლებს წვდომას შიდა გვერდებზე ლოგინამდე.

10. **e.preventDefault()**
    - **EN:** A built-in JavaScript method that cancels the default action that belongs to the event (such as preventing a form from submitting and reloading).
    - **KA:** JavaScript-ის ჩაშენებული მეთოდი, რომელიც აუქმებს ივენთის სტანდარტულ მოქმედებას (მაგალითად, აჩერებს ფორმის გაგზავნისას გვერდის რეფრესს).
